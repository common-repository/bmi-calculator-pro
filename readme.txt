== Description ==

This plugin calculates the BMI index according to the weights and height entered by the user. The plugin text fields and messages are in english but the user can translate/change all of these texts and also change the background color.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
